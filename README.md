
⏺ # VideoDash4 

An AI-powered SAT vocabulary video generator that creates short-form educational videos and publishes them directly to 
YouTube Shorts and TikTok. 

## What it does

1. You type a SAT vocabulary word (or leave it blank for a random one) 
2. Claude AI writes a cinematic scene script tailored to the word
3. OpenAI Sora 2 Pro generates the video in 9:16 vertical format 
4. You publish directly to YouTube Shorts and/or TikTok from the app 

## Features

- **AI scripted** — Claude designs the full video scene by scene 
- **Sora 2 Pro** — state-of-the-art video generation, native extension mode for longer videos
- **Extension chaining** — each scene extends the previous for a seamless continuous video 
- **Difficulty levels** — Easy / Medium / Hard / Expert vocabulary 
- **YouTube Shorts upload** — OAuth connected, metadata generated by AI
- **TikTok upload** — OAuth connected, direct post via Content Posting API 

## Requirements

- Python 3.11+
- Node.js 18+
- OpenAI API key with Sora access
- (Optional) Google Cloud project with YouTube Data API v3 for YouTube uploads 
- (Optional) TikTok Developer app with `video.publish` scope for TikTok uploads

## Setup

```bash 
# Install Python dependencies
pip install -r requirements.txt

# Install frontend dependencies
cd frontend && npm install && npm run build && cd ..

# Start the app
./start.sh

Open http://localhost:8001

Configuration

Go to Settings in the app to configure:

- OpenAI API Key — required for video generation 
- YouTube — paste path to your client_secret.json from Google Cloud Console, then connect your account
- TikTok — paste your Client Key and Client Secret from the TikTok Developer Portal, then connect your account 

Tech Stack 

- Backend — FastAPI + aiosqlite 
- Frontend — React + Zustand + Tailwind CSS
- AI — Claude CLI (scene scripting) + OpenAI Sora 2 Pro (video generation)
