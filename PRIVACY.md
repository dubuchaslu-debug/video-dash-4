# Privacy Policy 

This application is a personal tool for generating and publishing educational SAT vocabulary videos to TikTok via the
TikTok API. 

## Data Collection
This application does not collect, store, or share any personal data from TikTok users.

## TikTok API Usage
This app uses the TikTok Content Posting API solely to upload videos to the authenticated user's own TikTok account. No data is read from TikTok beyond what is required for authentication.

## Third-Party Services
Videos are generated using OpenAI's Sora API. No TikTok user data is passed to OpenAI.
